# 植物大战僵尸 v1.0

网页版植物大战僵尸单关卡游戏，纯前端、无任何外部依赖。

## 玩法

直接双击 `index.html` 即可游玩（无需任何服务器或安装）。

> 说明：`index.html` 加载已构建的 `dist/bundle.js`（普通 `<script>`，不受 `file://` CORS 限制）。`js/*.js` 是源模块，仅供开发维护。

- **收集阳光**：点击天空掉落或向日葵产出的阳光
- **种植**：选中底部植物卡片，再点击草坪格子种植
- **目标**：消灭全部 3 波僵尸即可胜利
- **小推车**：每行最左有 1 台，僵尸抵达左边时自动启动碾压（每行限用一次）

## 6 种植物

| 植物 | 阳光 | 用途 |
|---|---|---|
| 🌻 向日葵 | 50 | 每 24 秒产生 1 个阳光 |
| 🌱 豌豆射手 | 100 | 远程攻击同行僵尸 |
| 🌰 坚果墙 | 50 | 4000 HP 高血量肉盾 |
| 🍒 樱桃炸弹 | 150 | 1.2 秒后 3×3 范围秒杀 |
| ❄️ 寒冰射手 | 175 | 攻击 + 减速 50% 持续 4 秒 |
| 🥔 土豆雷 | 25 | 14 秒武装后接触爆炸秒杀 |

## 3 种僵尸

| 僵尸 | HP | 特点 |
|---|---|---|
| 普通僵尸 | 200 | 基础 |
| 路障僵尸 | 200 + 360 路障 | 头戴橙色路障 |
| 铁桶僵尸 | 200 + 1100 铁桶 | 头戴铁桶，最难对付 |

## 关卡节奏

- **第 1 波**（10 秒后）：5 只普通僵尸
- **第 2 波**（50 秒后）："一大波僵尸即将来袭！" 10 只混合僵尸
- **第 3 波**（105 秒后）：最终大波 15 只混合僵尸（含铁桶）

## 操作

- **桌面**：鼠标点击
- **移动端**：横屏 + 触屏点击
- **暂停**：右上角按钮

## 开发模式

URL 后加 `?dev`（如 `index.html?dev`）启用：
- `window.game` 暴露游戏状态
- 按键 `z`/`x`/`c` 分别召唤普通/路障/铁桶僵尸
- 按键 `s` 增加 100 阳光

## 技术细节

- HTML5 Canvas 2D 渲染
- 源代码用 ES Modules 组织；通过 `node build.js` 打包为单文件 `dist/bundle.js`
- Web Audio API 程序化音效
- 固定步长（60 Hz）游戏循环
- 模块化目录：`js/{config,game,grid,plants,zombies,projectiles,sun,particles,mower,renderer,ui,input,audio,levels,main}.js`

## 开发流程

修改源码后重新构建：

```bash
node build.js   # 重新生成 dist/bundle.js
```

然后刷新 `index.html` 即可看到效果。
